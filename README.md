## Run Chrome NoVNC
### Install curl if you don't have it
``` 
sudo apt install curl
```
### Run Chrome NoVNC using Curl
```
curl -fsSL https://raw.githubusercontent.com/GitXpresso/Waterfox-VNC/refs/heads/main/chromenovnc.sh | bash
```
